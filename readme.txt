NODE VERSION v16.13.0


=============INSTALL PACKAGE============
> Clone folder
> npm install

=============RUN SCRIPT============
> Open file balance.ts
> Pass parameter to function getMultipleBalance with three params
    - date (follow format: YYYY-MM-DD HH:mm:ss:SSS - Example: 2022-12-31 23:59:59:999)
    - timeZone (example: 9)
    - fileName (example: data.csv | Only support CSV export)
> Run commandline at root dictionary: ts-node balance.ts
